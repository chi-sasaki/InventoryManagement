### サービス概要
本プロジェクトは、板金加工工場を想定した在庫管理システムです。  
各製造工程で使用する部品・塗料などの資材や、完成した製品の在庫を一元管理することができます。  
また、入庫・出庫履歴を記録し、指定期間での使用量や入庫量を集計できる棚卸し機能も備えています。  

### 作成背景
板金加工工場に勤務していた際、担当工程で使用している大量の部品を、半年に一度棚卸しして在庫管理を行う必要がありました。当時はすべて手書きで管理していたため、棚卸し作業に2〜3日かかり、通常業務が滞ってしまうことがありました。

また、完成した製品の在庫管理を任されたこともありましたが、何百種類とある製品の在庫状況は、梱包作業を行う従業員の記憶に頼って管理されていました。  
そのため、「最後に注文があったのはいつ頃か」といった情報を確認する際にも、都度従業員に尋ねる必要があり、非常に非効率だと感じていました。

これらの課題を解決するため、部品・資材・製品の在庫情報や入出庫履歴を一元管理し、棚卸し作業を効率化できるアプリケーションが必要だと考え、本システムを作成しました。

### 機能一覧  
| 機能 | 詳細 |
| --- | --- |
| 資材（部品・材料）の一覧表示 | 登録されている全ての部品・資材情報を一覧で表示します |
| 資材の種類別検索 | プルダウンで部品・資材の種類を指定し、該当する情報を表示します |
| 資材の登録 | 部品・資材の名称、型番、数量、注文日を登録します |
| 資材の更新 | 一覧画面から更新画面に遷移し、資材情報を更新します |
| 資材の削除 | 在庫が残っていない場合のみ、資材情報を削除できます |
| 製品の一覧表示 | 完成した製品の在庫情報を一覧で表示します |
| 製品の名称検索 | 製品名を入力し、該当する製品情報を検索します |
| 製品の登録 | 製品の名称、型番、数量、最終注文日を登録します |
| 製品の更新 | 一覧画面から更新画面に遷移し、製品情報を更新します |
| 製品の削除 | 製品情報を削除します |
| 棚卸し機能 | 指定期間内の入庫数・出庫数を集計し、資材ごとの使用量を確認できます |
  
＊ 言葉の定義は以下の通りです  
**資材**  
各工程で使用する部品・材料を指します。  
圧入工程ではナット、塗装工程では塗料など、工程ごとに使用する資材が異なるため、工程単位で管理しています。  
**製品**  
各工程を経て完成した製品を指します。  
不良品や追加注文に備えて在庫を保持していたため、工程には依存しない独立したオブジェクトとして設計しています。  
**棚卸し機能**  
半年に一度実施される棚卸し業務を想定し、指定した期間内の入庫数・出庫数・使用量を集計、表示する機能です。
